# Lab 7 - Task 2: OOP Project

## Vehicle Class Hierarchy

- `Vehicle` — base class with brand, speed, fuel attributes
- `Car` — inherits Vehicle, adds doors attribute, overrides move()
- `Motorcycle` — inherits Vehicle, adds type attribute, overrides move()

## Concepts demonstrated
- Inheritance
- Method overriding
- Polymorphism
- Modules and imports

python main.py
